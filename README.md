# stable-matching

Stable Roommate Problem implemented in Javascript, interactive UI

[Also in Java](https://github.com/mdramos/stable-matching-java)


## Getting Started

### Installing

```
# after a pull just install deps
npm i 
```

### Running
```
npm start
```
